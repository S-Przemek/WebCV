Looker – MERN Social News & Microblog Platform

  • Designed and developed a full-stack MERN application combining social news, microblogging, and real-time messaging features.
  • Built a React + TypeScript (Vite) frontend and an Express / Socket.IO backend backed by MongoDB and Redis.
  • Implemented real-time functionality including direct messaging, typing indicators, presence tracking, and read receipts using WebSockets (Socket.IO).
  • Designed RESTful APIs for posts, comments, micro-posts, voting, bookmarks, search, and user profiles.
  • Integrated AWS S3 for scalable media storage (avatars, posts, comments, micro-posts, replies, and messages).
  • Used Redis for presence management, refresh token storage, and rate limiting of messaging actions.
  • Focused on scalable architecture, clean separation of client/server concerns, and maintainable code structure.

Chick – Native Desktop Chat Client for Kick.com

  • Designed and developed a native desktop chat client for Kick.com using C++17 and Qt6, inspired by the Chatty chat client for Twitch.
  • Implemented real-time chat overlays using WebSockets, displaying live messages directly on the streamer’s desktop.
  • Built audio notifications and optional Text-to-Speech (TTS) functionality to allow streamers to receive chat updates hands-free.
  • Focused on low-overhead, desktop-first design with native integrations such as system tray support and notifications.
  • Developed a lightweight Node.js helper service to handle OAuth redirects, token relay, and optional websocket bridging for authentication flows.
  • Structured the project as a multi-repo system, separating the native client, backend helper, and documentation/website.
